# Ejercicio práctico: ficha personal

## Instrucciones

Completa `exercise.py`.

El programa debe:

1. pedir el nombre;
2. pedir la edad;
3. pedir la ciudad;
4. mostrar una línea de separación;
5. mostrar el nombre y la edad usando argumentos separados por comas;
6. mostrar la ciudad usando una f-string;
7. mostrar los tres datos separados por ` | `.

Debes utilizar:

- `input()`;
- `print()`;
- una f-string;
- el argumento `sep`.

## Ejemplo

```text
Nombre: Ana
Edad: 20
Ciudad: Lima
--------------------
Nombre: Ana Edad: 20
Vive en: Lima
Ana | 20 | Lima
```
