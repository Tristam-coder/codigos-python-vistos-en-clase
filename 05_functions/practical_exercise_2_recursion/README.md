# Ejercicio práctico 2: recursión

## Instrucciones

Completa `exercise.py`.

Debes implementar dos funciones recursivas. Cada una necesita:

- un caso base que detenga la recursión;
- un caso recursivo que reduzca el problema.

## `countdown(number)`

Muestra los números desde `number` hasta `1`. Cuando llega a `0`, muestra
`"Fin"` y termina.

## `sum_to(number)`

Devuelve la suma de los números desde `number` hasta `0`.

Ejemplo:

```text
sum_to(4) = 4 + 3 + 2 + 1 + 0
```

## Salida esperada

```text
5
4
3
2
1
Fin
Suma de 1 a 5: 15
```
