# Ejercicio práctico: números y cadenas

## Instrucciones

Completa `exercise.py`.

El programa debe trabajar con un producto, su precio y la cantidad comprada.

Debes:

1. calcular el subtotal;
2. calcular cuántos paquetes completos de 3 unidades se pueden formar;
3. calcular cuántas unidades sobran;
4. unir el nombre y la categoría del producto;
5. obtener el primer carácter del nombre;
6. obtener los primeros cuatro caracteres;
7. convertir el nombre a mayúsculas;
8. reemplazar una palabra de la descripción;
9. dividir la descripción en palabras;
10. mostrar todos los resultados.

## Ejemplo de salida

```text
Subtotal: 52.5
Paquetes completos: 2
Unidades sobrantes: 1
Producto completo: Cuaderno - Escolar
Primer carácter: C
Primeros cuatro caracteres: Cuad
Nombre en mayúsculas: CUADERNO
Descripción modificada: Cuaderno para dibujo
Palabras: ['Cuaderno', 'para', 'clases']
```
