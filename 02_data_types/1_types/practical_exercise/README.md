# Ejercicio práctico: tipos de datos

## Instrucciones

Completa `exercise.py`.

Debes representar los datos de un curso utilizando el tipo indicado:

1. nombre del curso: `str`;
2. cantidad de estudiantes: `int`;
3. promedio del curso: `float`;
4. curso activo: `bool`;
5. temas del curso: `list`;
6. información del aula: `dict`;
7. fecha del examen: `NoneType`.

Después, muestra el valor y el tipo de cada variable usando `print()` y
`type()`.

## Ejemplo de salida

```text
Curso: Python | Tipo: <class 'str'>
Estudiantes: 25 | Tipo: <class 'int'>
Promedio: 15.5 | Tipo: <class 'float'>
Activo: True | Tipo: <class 'bool'>
Temas: ['consola', 'tipos', 'operadores'] | Tipo: <class 'list'>
Aula: {'numero': 4, 'piso': 2} | Tipo: <class 'dict'>
Fecha del examen: None | Tipo: <class 'NoneType'>
```

No se evalúa inmutabilidad en este ejercicio.
