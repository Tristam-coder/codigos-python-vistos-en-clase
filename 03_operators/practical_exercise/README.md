# Ejercicio práctico: operadores

## Instrucciones

Completa `exercise.py`.

El programa representa una compra y debe utilizar:

- operadores aritméticos;
- operadores de comparación;
- operadores lógicos;
- operadores de asignación;
- operadores bit a bit.

## Datos

- Precio: `12.5`
- Cantidad: `8`
- Dinero disponible: `120`
- Stock inicial: `20`

## Resultados esperados

```text
Subtotal: 100.0
Vuelto: 20.0
Cajas completas: 2
Unidades sobrantes: 2
Potencia: 8
¿Tiene dinero suficiente?: True
¿Es una compra grande?: True
¿El pago es exacto?: False
¿La compra está aprobada?: True
¿Requiere revisión?: False
Stock final: 14
AND bit a bit: 0b100
OR bit a bit: 0b1101
XOR bit a bit: 0b1001
Desplazamiento izquierdo: 0b11000
Desplazamiento derecho: 0b110
```

## Nota sobre operadores bit a bit

Los valores `0b1100` y `0b0101` están escritos en formato binario. La función
`bin()` permite mostrar el resultado también en formato binario.
